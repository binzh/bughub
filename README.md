#BugHub
----------
##目标
	打造一个高度自定义的缺陷管理系统

##功能点
###缺陷管理
* bug增改 √
* 自定义bug字段
	*  用户列表
	*  组列表
	*  文本
	*  数字
	*  枚举
	*  多选枚举
* bug状态变更（用户自定义缺陷变更流程） √  
* bug查询
* 变更历史

###系统管理
* 管理用户自定义内容
* 用户、组、权限管理
* 报表

###用户
* 用户信息
* 用户组
* 组权限
